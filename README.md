# Sega Model 3 UI
*Copyright 2024 PonMI The man eat shark*

## Overview

Sega model 3 UI is (Runs on Windows operating systems 10 or 11)　a launcher for [Supermodel](https://github.com/trzy/Supermodel), an emulator for Sega Model 3. Supermodel is a program that emulates the hardware of Sega Model 3, allowing users to run Sega Model 3 games on their computers. Sega Model 3 UI provides an interface to make Supermodel more user-friendly, enabling users to easily configure settings and launch and manage games.
